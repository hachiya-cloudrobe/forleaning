# 実行方法
* javac OthelloBanMain.javaして、java OthelloBanMain

# 操作方法
白と黒の手番が交互にくるので、コマンドプロンプトに >A:3 のようにマスを指定する。  
置けない場合は >p でパスできる。 >q で終了。
